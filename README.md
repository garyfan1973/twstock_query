# 台股即時報價查詢系統 📈

專業的台灣股票即時報價與技術分析系統，提供完整的技術指標圖表。

## 功能特色

### 即時報價
- 股票即時價格、漲跌幅
- 開盤價、最高價、最低價、昨收
- 成交量、本益比、52 週高低點

### 技術指標
- **K 線圖**：日 K 線含布林通道
- **移動平均線**：5日、10日、20日、季線(60日)、半年線(120日)、年線(240日)
- **KD 隨機指標**：超買超賣區標示
- **MACD**：DIF、MACD、柱狀圖
- **RSI 相對強弱指標**：超買超賣區標示
- **乖離率 (BIAS)**：5日、10日、20日乖離
- **威廉指標 (Williams %R)**

### 互動功能
- 時間區間切換：1月/3月/6月/1年/2年/5年
- MA 線顯示切換
- 所有圖表時間軸同步縮放
- 熱門股票快速查詢

## 安裝

```bash
pip install -r requirements.txt
```

## 執行

```bash
python app.py
```

預設在 http://localhost:5000 啟動

## 部署 (Gunicorn)

```bash
gunicorn -w 4 -b 0.0.0.0:8000 app:app
```

## 技術棧

- **後端**：Flask + yfinance + ta (技術分析)
- **前端**：Vanilla JS + Lightweight Charts
- **資料來源**：Yahoo Finance

## 股票代號格式

| 類型 | 代號範例 | 說明 |
|------|----------|------|
| 上市股票 | 2330 | 台積電 |
| 上櫃股票 | 6547 | 高端疫苗 (自動嘗試 .TWO) |
| ETF | 0050, 00878 | 元大台灣50、國泰永續高股息 |

## 注意事項

- 資料來源為 Yahoo Finance，約有 15-20 秒延遲
- 交易時間：週一至週五 09:00-13:30（台北時間）
- 非交易時間顯示上一交易日收盤資料
- 本系統僅供參考，投資有風險
